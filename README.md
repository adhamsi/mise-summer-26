# MISE — Introduction to Programming (Summer 2026)

Welcome! This is your **coding workspace** for the class. It lives in the cloud so you can access it from any computer without installing anything.

---

## What is this?

This is a **GitHub Codespace** — think of it as a computer that runs inside your web browser. It comes with Python and a code editor (VS Code) already set up and ready to use. You don't need to install anything on your own computer.

Every student has their own private copy of this workspace.

---

## How to open it

1. Go to [GitHub.com](https://github.com) and sign in
2. Open this repository and click the green **Code** button
3. Select the **Codespaces** tab and click on your codespace
4. The editor will open in your browser — you're ready to go!

> **Tip:** You can also open the codespace in the full VS Code desktop app by clicking the three-dot menu and selecting *"Open in VS Code Desktop"*.

---

## How to run a Python program

1. Open a file ending in `.py` by clicking it in the file explorer on the left
2. Press the **triangle (▶) button** in the top-right corner of the editor to run it
3. The output will appear in the panel at the bottom of the screen

> There is also a Terminal at the bottom of the screen — you don't need to worry about it for now.

---

## Folder structure

The workspace is organized by class. Each class has two folders:

```
class1/
    lecture/      ← programs we write together during class
    homework/     ← programs you need to write for homework
class2/
    lecture/
    homework/
...
```

The **lecture/** folder contains the programs from class, so you can open and run them yourself to experiment.

The **homework/** folder contains empty `.py` files — one per problem. Your job is to write the code inside them.

---

## Homework

After each class, you will be assigned homework. Here is how it works:

1. Open the corresponding `homework/` folder (e.g., `class1/homework/`)
2. You will find one empty `.py` file per problem
3. Write your solution inside the file and press ▶ to run it
4. Once it works, submit it on **PrairieLearn** at [pl.mise-camp-programming.xyz](https://pl.mise-camp-programming.xyz/) — the website will tell you if your solution is correct

> **Important:** Writing the code yourself is how you actually learn. Even if it's hard at first, keep trying — and ask for help on Piazza if you get stuck!

---

## Saving your work

Your work is saved automatically in the cloud. As long as you write code in this codespace, it will be there the next time you open it.

> **Warning:** If you write code outside this Codespace (for example, in a local file on your computer), it won't appear here. Always do your homework inside the Codespace.
